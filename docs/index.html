CPSC 332 Final Project Presentation

My Movies List

Ethan Bao

This is a movie list database where you can make an account and add movies to your list. You can give the movie a rating and your viewing status. You can also view details on all the movies in the database.

Functional Requirements:

  1. Users must be able to look through the list of movies within the database (front + back)

  2. Users must be able to search for movies (front + back)

  3. Users must be able to view the movie’s page filled with details such as when the movie released and its cast (front + back)

  4. Users must be able to create an account (front + back)

  5. Users must be able to add a movie to their movie list (front + back)

  6. Users must be able to rate a movie out of a scale of 10 (front + back)

  7. Users must be able to change their score and viewing status after adding to list (front + back)

  8. Users must be able to organize their movie list into categories of “Completed”, “Plan to Watch”, and “Dropped” (front + back)

  9. Users must be able to remove movies from their movie list (front + back)

  10. Users must be able to view other users’ movie lists (front + back)
